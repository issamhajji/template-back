# Instalar los siguientes paquetes:
- **Mongoose**
- **Dotenv**
- **Swagger**
- **CORS**
- **Nodemon** (solo para desarrollo)

```bash
npm install mongoose dotenv swagger-ui-express cors
````
````bash
npm install --save-dev nodemon
````
